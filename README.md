# 🚀 EEL Compiler

A compiler for the EEL programming language, developed as part of a university project.  
This project demonstrates core compiler construction concepts including lexical analysis, parsing, and intermediate code generation.

---

## 📌 Overview

The **EEL Compiler** processes source code written in the EEL language and translates it into intermediate representation.

It was built to simulate the main phases of a real-world compiler:

- Lexical Analysis (Tokenization)
- Syntax Analysis (Parsing)
- Intermediate Code Generation
- Error Detection & Handling
